# clj4tezos

This is a demo of using the `ej4tezos` libraries with Clojure in order to bring some Lispy goodness to the Tezos universe.

## Prerequisites

You will need `lein` and `mvn` installed (`apt install maven leiningen`)

then

`lein deps; lein repl`
